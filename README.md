# AI-POWERED INTELLIGENT DATA ANALYTICS ASSISTANT

## Overview

AI-Powered Intelligent Data Analytics Assistant is a full-stack data analytics application developed using Python, Flask, Pandas, Scikit-learn, SQLite, and Matplotlib. The system performs data analysis, visualization, machine learning model training, AI-generated insights, and automated report generation using the Titanic dataset.

## Key Features

* Dataset Analysis and Summary
* Data Visualization Dashboard
* Random Forest Machine Learning Model
* AI-Based Insights Generation
* Automated Analytics Reports
* SQLite Database Integration
* Interactive Web Interface using Flask

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* SQLite
* Joblib

## Project Outcome

* Performed Exploratory Data Analysis (EDA)
* Generated Multiple Visualizations
* Trained a Random Forest Classifier
* Achieved 72.07% Prediction Accuracy
* Generated AI Insights and Reports
* Stored Analytics Results in SQLite Database
